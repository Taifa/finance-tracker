# Finance Tracker

A vanilla JavaScript finance tracker that supports CRUD, category filtering, filtered totals, and localStorage persistence.

## Run

The app is live on GitHub Pages here:
https://taifa.github.io/finance-tracker/

## Screenshot 
![alt text](<Screenshot 2026-04-23 at 23.41.24.png>)

## Features

- Add expenses
- Edit existing expenses
- Delete expenses
- Assign expenses to categories
- Filter expenses by category
- Display total based on active filter
- Persist expense data with localStorage

- HTML

- JavaScript
